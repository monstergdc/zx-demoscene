
This is collection of four ZX Spectrum 48 kB demos from 1990
created by MoNsTeR of GDC (Jakub Noniewicz).
They were recovered from old tapes 20 years later, in 2010.
They work fine with Spectaculator 7.x ZX Spectrum emulator.
Four file types are provided for each demo: *.tap, *.z80, *.tzx, *.sna.

Rev.: 2010.05.27
Rev.: 2011.06.10

Demos include:
===============
- GIGA DEMO
- SHORT DEMO
- SPECCY IMAGES I
- SPECCY IMAGES II
